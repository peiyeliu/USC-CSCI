To compile the program, type:
    javac ImageDisplay2.java

Put rgb input files in the same folder

To run the program, type:
    java ImageDisplay2 [RGB file] Q M L

Q: quantization level, should be an integer from 0 to 7
M: delivery mode, should be an integer ranging from 1, 2, 3
L: latency, unit is milliseconds, it should be an integer
